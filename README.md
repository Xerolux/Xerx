# Xerx

A clean and minimal WordPress theme for documenting your stream of life.

Xerx is a minimal yet extensive WordPress theme. It has different styles for post types and allows you to customize colors, typography, spacing and images. It supports the block editor with custom block patterns, dark mode, SEO enhancements, breadcrumbs, social sharing, reading time estimates, related posts, and portfolio layouts via Jetpack.

## Features

* **Minimal Design**: Clean, minimal design with multiple post format styles.
* **Dark Mode**: Built-in dark mode support with OS preference detection.
* **Customization**: Extensive customizer options for colors, typography, spacing, and layout.
* **SEO Enhancements**: Built-in SEO meta tags, JSON-LD schema, breadcrumbs, and canonical URLs.
* **Social Sharing**: Social sharing buttons with copy-link functionality.
* **Engagement**: Reading time estimates and related posts support.
* **Portfolio**: Portfolio grid layout support via Jetpack integration.
* **Block Editor Ready**: Full block editor support with custom block patterns and editor style parity.
* **Responsive**: Fully responsive design with mobile-friendly navigation.

## Requirements

* **PHP**: 7.4 or later
* **WordPress**: 6.4 or later (tested up to 7.0)

## Installation

1. Download the theme as a `.zip` file from the repository.
2. Log into your WordPress site and navigate to **Appearance > Themes**.
3. Click the **Add New** button at the top, then **Upload Theme**.
4. Choose the downloaded `.zip` file and click **Install Now**.
5. Once installed, click **Activate** to start using the Xerx theme.

## Customization

Navigate to **Appearance > Customize** to access the theme settings:

* **Colors**: Change the header background, foreground color, and dark mode specific colors.
* **Typography**: Select from various Google Fonts for headings, body text, and monospace elements. You can also disable remote fonts to use system fallbacks.
* **Spacing**: Adjust the main content width, line height, and base font size.
* **Layout Options**: Toggle the footer colophon and comment avatars.

## Post Formats

Xerx supports different post formats, allowing you to style your content differently depending on the type of post:

* **Standard**: Default blog post style.
* **Aside**: Designed for quick, shorter updates.
* **Status**: Displays content with a larger font, ideal for short status updates.
* **Image**: Optimized for posts consisting primarily of a single image.
* **Chat**: Formatted specifically for chat transcripts or dialogs.

## Jetpack Portfolio

If you use the Jetpack plugin, Xerx offers specialized templates for the Portfolio custom post type (`archive-jetpack-portfolio.php` and `single-jetpack-portfolio.php`), presenting your projects in a grid layout.
